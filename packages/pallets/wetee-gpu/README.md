# ***WETEE Module***

***
## Overview
App module for user to deploy and manage TEE app.

***
## All Calls
***
* `create`  - Create tee app.
* `update`   - Update tee app.
* `set_settings`   - Set tee app public evnironment config.
* `recharge`   - Recharge tee app.
* `stop`     - Stop running tee app.
